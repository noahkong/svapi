 Català
--------

1. Introducció.

SVAPI és una llibreria d'estereovisió capaç de processar estereoscòpicament imatges obtingudes a partir de webcams, obtenint així mapes de disparitat i transformació de coordenades. DemoEstereo és un programa de prova capaç de demostrar totes aquestes capacitats.

2. Instruccions.

Per compilar, obtenir Dev-C++ (http://www.bloodshed.net/devcpp.html) i obrir el projecte associat. Poden ser necessàries aquestes llibreries: 
-Gtk+
-Glade
-VideoInput
-OpenCV

Per provar DemoEstereo, simplement executar l'arxiu "DemoEstereo.exe". Consultau "config.ini" per configurar l'aplicació. Són necessàries dues webcams connectades a l'ordinador per utilitzar el mode de captura. Es recomana connectar aquestes a diferents concentradors USB per evitar problemes hardware.

3. Crèdits.

Part del projecte "Sistema estereoscòpic de baix cost" realitzat per Víctor Blanch.
Director: Antoni Jaume i Capó.
Universitat de les Illes Balears 2007-2008.

 English
------------

1. Introduction.

SVAPI is a stereovision library able to process stereoscopically images from webcams, obtaining disparity maps and making possible coordinate transformation. DemoEstereo is a testing program able to demonstrate all this capabilities.

2. Instructions.

To compile, first obtain Dev-C++ (http://www.bloodshed.net/devcpp.html) and open the associated project. Several libraries might be needed:
-Gtk+
-Glade
-VideoInput
-OpenCV

To test DemoEStereo, simply execute the file "DemoEstereo.exe". Check "config.ini" to configure the app. Two webcams connected to the computer are needed to use the capture mode. It's recommended to connect them to different USB concentrators to avoid hardware problems.
